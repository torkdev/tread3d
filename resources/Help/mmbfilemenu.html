MAIN MENU BAR - FILE MENU



File Menu


New This will allow you to initiate a new map base upon which .GDF2 script you choose.

Open This will allow you to import a .TRD file.

Close This will allow you to close your map.

Import Map File This will allow you to import a .MAP file.

Export Map File This will allow you to export a .MAP file.

Save This will allow you to save your map as a .TRD file.

Save As This will allow you to save your map under a new name as a .TRD file.

1 This block lists all the previously opened files.

Exit This will exit you from Tread.
