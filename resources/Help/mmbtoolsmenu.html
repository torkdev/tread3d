MAIN MENU BAR - TOOLS MENU



Tools Menu


Flip Horizontal This will flip the current selection in the horizontal direction.

Flip Vertical This will flip the current selection in the vertical direction.

Rotate 90 CW This will rotate the current selection 90 degrees clock-wise.

Rotate 90 CCW This will rotate the current selection 90 degrees counter-clock-wise.

Rotate 180 This will rotate the current selection 180 degrees.

Group This will allow you to group selected objects.

Un-Group This will allow you to breakup a selected group.

Carve This will allow you to carve a brush with a selected brush.

Hollow This will allow you to hollow a selected brush in a new window.

Link Entity to Brush This will allow you to link an entity to a selected object.

Unlink Entity from Brush This will allow you to unlink the current linked brush.

Entity Properties This will allow you to change the current entities properties in a new window.

Make Entity Path This will allow you to generate an open-ended entity path.

Make Closed Entity Path This will allow you generate a closed entity path.

Make Entity Targets This will allow you to target a selection as an entities target.

Delete Path This will allow you to delete an entities path.

Build Model This will allow you to save something you have built as a model/prefab.
