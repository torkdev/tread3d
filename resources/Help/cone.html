CREATING A CONE

This is a walk-through as to how to use the Shapes Plugin for Cones.

Before starting this tutorial you should first get acquainted with the basics of the "Tread3D" editor; the screen and the various commands that are available.

As in the previous tutorials start "Tread3D".
Lets make a very basic room that is 512x512x256 units.
Also load a different texture in the Texture Window than the ones used to build the room.

Your screen should look like the one below.


Now click on the "Plugins" menu and scroll down to "Shapes".


You will see three choices and these are:

Make Arch (This will automatically create a arch)
Make Cone (This will automatically create a cone)
Make Cylinder (This will automatically create a cylinder)


Lets point to the "Make Cone" option.

A new window should now appear.

The "Create Cone" window should look like the one below.


Now some important notes:
The "Axis" input determines what axis the center line of the cone will be on.
The "Length" input determines the overall height of the cone.
The "Cone Size" input determines what the diameter of the cone will be.
The "Number Of Sides" input determines the number of "Faces" that will make up the cone.

Let's change the "Axis" to "Z" and use the default settings for the other inputs.
Just click "Create".

Your screen should look like the one below.


Now lower the cone so that it touches the floor of your room.

Your screen should look like the one below.


Now some important notes:
The "Cone" imported into your map has the original characteristics that you inputted.
The "Cone" is generated as a single "Brush".
The larger the "Number Of Sides" the smoother the cone will appear.


Thats all there is to it.

What you have done is "Imported a cone using the shapes plugin".


Now resave your work.
