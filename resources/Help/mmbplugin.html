MAIN MENU BAR - PLUGIN MENU



Plug-In Menu


Plugins This will allow you load a plug-in.




Leak Trace This will allow you load a plug-in that depicts leaks in the 3D View.


Load Leak File This will allow you to browse for the map (.Lin) file.



Zombielight This will allow you load a plug-in that depicts actual lighting in the 3D View.


Show Light This will allow you to see only light from light entities in the 3D View.

Shadows This will allow you to see light shadows in the 3D View after you initiate a "Preview".

Build Light Preview This will allow you to see full lighting effects in the 3D View.



Shapes This will allow you load a plug-in that builds shapes automatically.


Arch This will allow you to create an arch.

Cone This will allow you to create a cone.

Cylinder This will allow you to create a cylinder.
