OBJECT TOOLBAR



Group This will allow you to group selected objects.

Ungroup This will allow you to breakup a selected group.

Hollow This will allow you to hollow a selected brush in a new window.


Carve This will allow you to carve a brush with a selected brush.

Flip Vertical This will flip the current selection in the vertical direction.

Flip Horizontal This will flip the current selection in the horizontal direction.

Rotate 90 CW This will rotate the current selection 90 degrees clock-wise.

Rotate 90 CCW This will rotate the current selection 90 degrees counter-clock-wise.

Rotate 180 This will rotate the current selection 180 degrees.

Link To Entity This will allow you to link an entity to a selected object.

Unlink This will allow you to unlink the current linked brush.

Entity Properties This will allow you to change the current entities properties in a new widow.
