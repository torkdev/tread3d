OBJECT WORKSPACE



Object Workspace Window This is the Object Workspace display window.

Arrow Button This will increase the size of the Object Workspace display window.

X Button This will minimize the size of the Object Workspace.

Visgroups Button This will display all Visgroups in the Object Workspace display window.

Anchors Button This will display all Anchors (positionable view references) in the Object Workspace display window.

Objects Button This will display all Objects (Brushes and Models) in the Object Workspace display window.
